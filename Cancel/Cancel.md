# Cancel

## Cancel Date
- It's the date which indicates that the policy is cancelled. It will be clearly labelled in the document. (When the Document ID is NON Renewal, you must use Non-Renewal Date as Cancel Date, otherwise it is blank).

## Cancel reason 
- This field has a present list of values and this field MUST be filled in on all cancels.

### NON PAY:

NON PAY/similar for we have not received the payment.

### OTHER:

If no reason is listed, choose other.

### REQUEST:

REQUEST.

### U/W:

Stands for underwriting reasons.
