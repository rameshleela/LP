# Policy No
- Policy Number
- Reference Number
- File Number
- Quote Number
- Certificate Number
- Binder Number
- Master Policy Number
- Application Number
- Offer Number

## Rules
- We can take the policy number as per the document.
- If the **policy number** contains only **alphabets** we should add **"Space 1"** at the end.
- If the **policy number is not available** in the document we can take it as **Binder**.
- For the **Quote** and **Application** the policy number is **not required**.
