# Insured Name
- Policy holder
- Customer
- Borrower
- Applicant

## Types
- Individual names
- Commercial names
- Association, Condominium, Condo, Assn, Town, Villa, HPR, COA, POA

## Rules
- As per the document.
- Mr, Mrs,Ms, Dr Should not take in Insured name.
- Single, Married, Widow, Tenant also shold not capture in Insured name.
- We do not capture insured name after the c/o and ATTN.
- We can take after DBA, AKA, FKA.
- We can take Jr, Sr, in Insured name.
- If insured name is not available in the document we can take it as No Name.
- If insured name contains Association, Condominium, Condo, Assn, Town, Villa, HPR, COA, POA then we can take policy type as HOA.

**DBA:** Doing business as

**FKA:** Formally known as

**AKA:** Also known as

## Examples

**Scott revocable trust 12/08/2025**
**Scott trustee**
- We can take Scott revocable trust.
- Don't take date.
- In second line Take only individual name that is Scott.

**Elizabeth 1234567 Sarah**
- We can take Individual names only as Elizabeth Sarah, don't take numbers if it has individual names.

**Ram insurance 12345 llc**
- We can take full insured name with numbers because it is commercial name.
